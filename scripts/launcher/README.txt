WealthClaude Launcher
=====================

Double-click any file to run:

  START - All Processors.vbs    → Runs Instagram + YouTube in background
  START - Reels Only.vbs        → Runs only Instagram reels
  START - YouTube Only.vbs      → Runs only YouTube videos
  STOP  - All Processors.vbs    → Kills everything
  VIEW  - Logs.bat              → Shows latest processor log

Notes:
  - Background scripts run even after closing the window
  - Processors auto-sleep from 11 PM to 7 AM
  - Logs saved in: scripts/logs/
